# Rust Lang

> <https://www.rust-lang.org>

```bash
# install
curl https://sh.rustup.rs -sSf | sh

# demo
cargo new hello_world
cargo run

# version
rustc --version

# upgrade
rustup update

# uninstall
rustup self uninstall
```

## Cargo (dependency manager and build tool)

> <https://doc.rust-lang.org/cargo/>

```bash
# format code
cargo fmt

# create package
cargo new hello_world

# compile package
cargo build

# compile and run
cargo run

# install dependency
cargo install wasm-pack
```
