# fractal/分形

1. [Context Free](https://www.contextfreeart.org), 可编程制作图形.
2. [Apophysis](http://www.apophysis.org).
3. [Fractint](https://fractint.org/).
4. [Incendia Next](https://www.incendia.net).
5. [Ultra Fractal](https://www.ultrafractal.com), 收费.
