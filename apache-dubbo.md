# Apache Dubbo

> <http://dubbo.apache.org>  
> Apache Dubbo™ (incubating)是一款高性能Java RPC框架。

| 节点 | 角色说明 |
| --- | --- |
| Provider | 暴露服务的服务提供方 |
| Consumer | 调用远程服务的服务消费方 |
| Registry | 服务注册与发现的注册中心 |
| Monitor | 统计服务的调用次数和调用时间的监控中心 |
| Container | 服务运行容器 |
